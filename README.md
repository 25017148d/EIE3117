# 3117project (RideShare)

This is a carpooling platform developed with Vue 3 (Frontend) and Django (Backend).

## Getting Started

### Prerequisites
- Node.js & npm
- Python 3.x

### Setup Frontend
```bash
npm install
npm run dev
```

### Setup Backend
```bash
python3 -m venv .venv
source .venv/bin/activate  # On Windows use: .venv\Scripts\activate
pip install -r requirements.txt
python backend/manage.py runserver
```
